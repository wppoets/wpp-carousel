== Description ==

Adds a slideshow content type to your WordPress installation. With an easy to use interface you can add a slides to a carousel for displaying on your website.

== Installation ==

1. Place the 'wpp-slideshow' folder in your '/wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit 'Settings' to adjusting the configuration.
